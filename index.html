Contoh Dashboard Cuaca Jawa Barat

This dashboard is part of my exercise in developing using Quarto Dashboards, publishing on Github Pages, and then automatically updated it using Github Actions.

Data and Packages

The input data for this dashboard is downloaded using a Weather API which acts as a (unofficial) weather API wrapper from XML to JSON references from API of BMKG(Badan Meteorologi, Klimatologi, dan Geofisika).

The dashboard uses the following packages:

  • Data visualization
    • highcharter
  • Utility
    • dplyr
    • tidyr
    • lubridate
    • janitor
